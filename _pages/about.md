Java/Spring 기반 백엔드와 C/C++ JNI 연동을 주로 다루는 3년차 개발자 원주연입니다.

계정·비밀번호 관리, 암호화와 무결성, Windows/Linux 운영 자동화, 네트워크 장비 API 연동을 구현해 왔습니다. 외부 장비나 네이티브 라이브러리처럼 애플리케이션 밖의 상태가 얽힌 문제를 로그와 코드로 나눠 확인하는 과정에 관심이 많습니다.

개인적으로는 Go, Python, Qt/C++를 사용해 장비 진단 도구, MPEG-TS 분석 도구, RAG 검색 실험, 개발 세션 아카이브 도구를 만들고 있습니다.

이 블로그에는 구현한 기능, 실패한 접근, 오류를 좁힌 과정, 다시 사용할 수 있는 기술 개념을 기록합니다.
